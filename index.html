Architectural Highlights

Hybrid & Multi-Cloud Connectivity

  • Architected 10Gb AWS Direct Connect circuits for multi-account landing zones, ensuring strict alignment with the AWS Well‑Architected Framework.
  • Designed Azure hub-and-spoke topologies featuring VNet peering, custom routing, and UDRs for ExpressRoute subscribers, guaranteeing consistent enterprise connectivity.
  • Engineered secure cross-cloud federation through IPsec VPN with BGP integration for frictionless communication between AWS and Azure ecosystems.

Infrastructure as Code & Automation

  • Championed NetDevOps adoption by authoring advanced CloudFormation templates, Terraform modules, and ARM/Bicep deployments to automate infrastructure at scale.
  • Automated AWS Transit Gateway attachment orchestration, centralizing routing and security protocols for complex multi-VPC architectures.
  • Constructed robust CI/CD pipelines utilizing AWS Developer Tools (CodeCommit, CodeDeploy, CodePipeline) to provision immutable infrastructure.

Identity, Zero-Trust Security & FinOps

  • Architected comprehensive cross-cloud IAM utilizing Entra ID, Azure RBAC, and AWS IAM Identity Center under a unified governance and MFA model.
  • Implemented NSX-T microsegmentation and zero-trust network boundaries mapped directly to the NIST Cybersecurity Framework (CSF) 2.0.
  • Developed FinOps intelligence dashboards utilizing advanced KQL and Log Analytics to eliminate cloud waste, optimize DTO costs, and execute showback/chargeback reporting.
